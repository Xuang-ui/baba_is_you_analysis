# Readme

## Baba Readme (by Yangfan)
- Each unique entity which occupies a grid position is called a "block".
- Each block has a unique identifier (property of `Block` class defined in [./pybaba/logic/space.py](./pybaba/logic/space.py)).

## cogalg
- To visualize the three types of graphs, use run_map_editor.py.
- Recommended usage: `python run_map_editor.py base --distance-mode path_through_empty` ("base" may be replaced by other map names)
- See map_editor_readme.md for details.
